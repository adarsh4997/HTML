Sematic tags

In HTML5, semantic tags are used to give meaning to the content, making it more understandable for both developers and browsers. These tags improve accessibility, SEO, and code readability. Here are some key semantic tags:

  • nav: Contains navigation links.
  • header: Defines the introductory content or navigation links.
  • main: Represents the main content of the document.
  • article: Represents a self-contained piece of content, like a blog post or news article.
  • section: Groups related content together.
  • aside: Contains content related to the main content but separate from it, like a sidebar.
  • footer: Defines the footer of a document or section, typically containing metadata or links.
  • figure: Encapsulates media content, like images or diagrams, with an optional caption.
  • figcaption: Provides a caption for the <figure> element.
